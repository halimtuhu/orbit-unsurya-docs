Pelaporan Penelitian dan PKM Mandiri

Fitur ini memungkinkan Anda melaporkan kegiatan penelitian atau pengabdian kepada masyarakat (PKM) yang dilakukan secara mandiri. Laporan akan diverifikasi oleh Admin LPPM dan disetujui oleh Ka. LPPM.

Mengakses Fitur​

Klik menu Pelaporan Penelitian dan PKM Mandiri di sidebar untuk melihat daftar laporan Anda.

Daftar Kegiatan Mandiri​

Halaman daftar menampilkan semua laporan kegiatan mandiri milik Anda. Jika belum ada laporan, halaman akan menampilkan pesan "Belum ada laporan kegiatan mandiri."

Pencarian dan Filter​

FiturFungsi
Cari judul kegiatanKetik kata kunci untuk mencari berdasarkan judul
Filter TahunPilih tahun kegiatan untuk menyaring laporan
Filter JenisPilih Penelitian, PKM, atau Semua Jenis
Filter StatusPilih status laporan (Draft, Diajukan, Perlu Revisi, Terverifikasi, Disetujui, Ditolak)

Membuat Laporan Baru​

  1. Klik tombol Buat Laporan di halaman daftar
  2. Formulir laporan akan terbuka dengan beberapa bagian yang harus diisi

Informasi Dasar​

Jenis Kegiatan (wajib)

Pilih jenis kegiatan:

  • Penelitian — untuk kegiatan riset mandiri
  • PKM — untuk kegiatan pengabdian kepada masyarakat

Skema (wajib)

Setelah memilih jenis kegiatan, pilih skema yang sesuai.

Skema untuk Penelitian:

  • PDP — Penelitian Dosen Pemula
  • PFR — Penelitian Fundamental Riset
  • PT — Penelitian Terapan
  • PP — Penelitian Pengembangan
  • Lainnya — jika skema tidak tersedia

Skema untuk PKM:

  • PMB — Pengabdian Masyarakat Berbasis
  • PBK — Pengabdian Berbasis Kemitraan
  • PBW — Pengabdian Berbasis Wirausaha
  • Lainnya

Judul Kegiatan (wajib)

Masukkan judul kegiatan penelitian atau PKM Anda.

Fakultas (wajib)

Pilih fakultas tempat Anda bertugas.

Program Studi (wajib)

Pilih program studi berdasarkan fakultas yang dipilih.

Tim Pelaksana​

Ketua Tim — nama Anda (otomatis dari profil, tidak dapat diubah)

Anggota Tim — Anda dapat menambahkan anggota dengan klik Tambah Anggota.

Dokumen​

Unggah dokumen pendukung kegiatan. Dokumen bertanda (*) wajib diunggah sebelum mengirim laporan.

Cara Upload:

  1. Klik area upload atau tombol Choose File
  2. Pilih file dari perangkat Anda
  3. File akan diunggah otomatis
  4. Nama file akan ditampilkan setelah berhasil diunggah
Perhatian

Laporan hanya dapat dikirim jika ketiga dokumen wajib sudah diunggah: Surat Tugas, Proposal, dan Laporan Kegiatan.

Menyimpan Laporan​

Simpan Draft — Menyimpan laporan sebagai draft tanpa mengirim. Anda dapat melanjutkan editing nanti.

Kirim untuk Verifikasi — Menyimpan dan mengirim laporan ke Admin LPPM. Status akan berubah menjadi Diajukan.

Batal — Kembali ke daftar laporan tanpa menyimpan.

Mengelola Laporan​

Lihat Detail​

Klik baris laporan di daftar untuk melihat halaman detail lengkap dengan semua informasi yang telah Anda masukkan.

Edit Laporan​

Edit hanya tersedia untuk laporan berstatus Draft atau Perlu Revisi.

  1. Buka halaman detail laporan
  2. Klik tombol Edit
  3. Form akan terbuka dengan data yang sudah terisi
  4. Ubah informasi yang diperlukan
  5. Klik Simpan Perubahan atau Batal
Catatan

Jika laporan berstatus Perlu Revisi, catatan dari Admin LPPM akan ditampilkan sebagai panduan perbaikan.

Hapus Laporan​

Hapus hanya tersedia untuk laporan berstatus Draft.

  1. Klik baris laporan atau tombol aksi
  2. Klik Hapus
  3. Konfirmasi bahwa laporan akan dihapus selamanya
  4. Klik Hapus untuk menyelesaikan
Perhatian

Laporan yang dihapus tidak dapat dikembalikan.

Status Laporan​

Laporan Anda akan melewati beberapa status sesuai dengan alur persetujuan:

StatusArtiAksi Dosen
DraftLaporan sedang disusun, belum dikirimEdit, Hapus, Kirim untuk Verifikasi
DiajukanLaporan sudah dikirim ke Admin LPPM, menunggu verifikasiLihat detail saja
Perlu RevisiAdmin LPPM meminta perubahan, baca catatan revisiEdit, Kirim Ulang
TerverifikasiAdmin LPPM sudah verifikasi, menunggu persetujuan Ka. LPPMLihat detail saja
DisetujuiKa. LPPM sudah menyetujui laporanLihat detail saja
DitolakKa. LPPM menolak laporan, baca alasan penolakanLihat detail saja

Mengirim Laporan​

Pertama Kali (dari Draft)​

  1. Buka halaman detail laporan berstatus Draft
  2. Pastikan ketiga dokumen wajib sudah diunggah
  3. Klik Kirim untuk Verifikasi
  4. Konfirmasi pengiriman
  5. Klik Ya, Kirim

Status akan berubah menjadi Diajukan dan Admin LPPM akan menerima notifikasi.

Kirim Ulang (setelah Perlu Revisi)​

  1. Buka halaman detail laporan berstatus Perlu Revisi
  2. Baca catatan revisi dari Admin LPPM
  3. Klik Edit untuk memperbaiki laporan
  4. Lakukan perubahan dan klik Simpan Perubahan
  5. Kembali ke halaman detail
  6. Klik Kirim Ulang
  7. Konfirmasi dengan klik Ya, Kirim Ulang

Status akan berubah kembali menjadi Diajukan.

Riwayat Status​

Bagian bawah halaman detail menampilkan timeline riwayat perubahan status laporan, termasuk:

  • Status awal dan status tujuan
  • Catatan dari Admin atau Ka. LPPM (jika ada)
  • Nama pengguna yang melakukan perubahan
  • Tanggal dan waktu perubahan

Timeline ditampilkan dari perubahan terbaru di atas.

Tips dan Panduan​

Informasi Penting
  • Nama Ketua Tim — diambil otomatis dari profil dosen Anda dan tidak dapat diubah
  • Dokumen PDF — pastikan file PDF dalam kondisi baik dan dapat dibaca
  • Kemitraan PKM — jika kegiatan adalah PKM, wajib mencantumkan nama mitra
  • Notifikasi — Anda akan menerima notifikasi setiap kali status laporan berubah
