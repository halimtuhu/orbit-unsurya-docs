# Membuat Proposal (7-step wizard)

Panduan lengkap untuk membuat proposal penelitian atau pengabdian kepada masyarakat melalui wizard 7 langkah di ORBIT UNSURYA.

## Memulai Pembuatan Proposal

1. Klik menu **Proposal** di sidebar
2. Pada halaman "Proposal Saya", klik tombol **Buat Proposal**
3. Anda akan diarahkan ke halaman pemilihan program dan batch

## Langkah 0: Pilih Program dan Batch

Sebelum memasuki wizard, Anda harus memilih program penelitian dan batch pengajuan.

![Pilih Program dan Batch](/img/dosen/proposal-create-step0-select-program.png)

### Memilih Program

1. Lihat daftar program penelitian yang tersedia di bagian "1. Select Program"
2. Setiap program menampilkan informasi:
   - Nama program
   - Jenis penelitian (Riset Dasar, Riset Terapan, dll)
   - Kategori (Penelitian, Pengabdian)
   - Dana maksimal
   - Jumlah batch aktif
3. Klik pada program yang sesuai dengan penelitian Anda
4. Program yang dipilih akan menampilkan tanda "Program selected"

### Memilih Batch

Setelah memilih program, batch yang tersedia akan ditampilkan di bagian "2. Select Batch".

1. Lihat daftar batch yang tersedia
2. Setiap batch menampilkan:
   - Nama batch
   - Deadline pengajuan
   - Sisa hari untuk pengajuan
   - Jumlah proposal yang sudah masuk
3. Klik batch yang ingin Anda gunakan
4. Batch yang dipilih akan menampilkan tanda "Batch selected"

### Lanjut ke Wizard

Setelah memilih program dan batch, tombol **Continue to Proposal Details** akan aktif.

![Batch Terpilih](/img/dosen/proposal-create-step0-batch-selected.png)

Klik tombol tersebut untuk masuk ke wizard 7 langkah.

:::warning Perhatian
Setelah Anda memilih program dan batch, tidak dapat mengubahnya selama proses pembuatan proposal. Pastikan pilihan sudah benar sebelum melanjutkan.
:::

## Langkah 1: Identitas Usulan

Langkah pertama adalah mengisi identitas dan informasi dasar proposal penelitian atau pengabdian.

![Identitas Usulan](/img/dosen/proposal-create-step1-identitas-usulan.png)

### 1.1 Identitas Penelitian

**1. Judul Proposal (Wajib)**
- Masukkan judul proposal yang jelas dan deskriptif
- Judul harus mencerminkan inti dari penelitian atau pengabdian yang akan dilakukan

### 1.2 Pemilihan Program Penelitian

**1. Bidang Fokus Penelitian (Wajib)**
- Pilih bidang fokus penelitian dari dropdown
- Opsi meliputi: Kedirgantaraan, Teknologi dan Rekayasa, Ekonomi dan Bisnis, Kesehatan dan Masyarakat, Pendidikan dan Sosial

**2. Tahun Pertama Usulan (Wajib)**
- Masukkan tahun dimulainya penelitian (minimum tahun 1970)
- Format: YYYY (contoh: 2025)

**3. Ruang Lingkup (Wajib)**
- Pilih dari dropdown: Nasional, Internasional, atau Lokal
- Ini menunjukkan jangkauan/skala penelitian Anda

**4. Lama Kegiatan (Wajib)**
- Durasi penelitian sudah terisi otomatis (default: 1 tahun)
- Dapat diubah sesuai kebutuhan

**5-7. Rumpun Ilmu (Wajib)**
- **Level 1 (Rumpun Ilmu)**: Pilih rumpun ilmu umum terlebih dahulu
- **Level 2 (Sub Rumpun)**: Akan aktif setelah Level 1 dipilih
- **Level 3 (Bidang Ilmu)**: Akan aktif setelah Level 2 dipilih
- Hierarki ini memastikan klasifikasi penelitian yang tepat sesuai taksonomi universitas

**Taksonomi Bidang Fokus (Opsional)**
- Bagian tambahan untuk memperdalam klasifikasi penelitian
- Mencakup: Bidang Fokus Penelitian, Tema Riset, Topik Riset
- Hubungan antar field sama dengan Rumpun Ilmu - bersifat bertingkat

### 1.3 Identitas Ketua

**Nama Ketua**
- Nama otomatis terisi dari profil Anda
- Dapat diubah jika diperlukan

**Uraian Tugas dalam Penelitian (Wajib)**
- Jelaskan peran dan tanggung jawab Anda sebagai ketua penelitian
- Contoh: "Mengkoordinasikan dan mengelola keseluruhan pelaksanaan penelitian"

### 1.4 Identitas Anggota Penelitian (Dosen)

Bagian untuk menambahkan dosen yang terlibat dalam penelitian.

**Menambah Anggota**
1. Klik tombol **+ Tambah**
2. Akan tampil form untuk memasukkan data anggota dosen:
   - NIDN (Nomor Induk Dosen Nasional)
   - Nama
   - Tugas/peran dalam penelitian
3. Setelah mengisi, anggota akan muncul dalam tabel

**Tabel Anggota**
- Menampilkan kolom: No, NIDN, Nama, Tugas, Status, Aksi
- Status menunjukkan verifikasi anggota
- Kolom Aksi untuk mengubah atau menghapus anggota

:::tip Catatan
Anggota dosen harus merupakan dosen Universitas Dirgantara Marsekal Suryadarma yang terdaftar dalam sistem.
:::

### 1.5 Identitas Anggota Non Dosen Unsurya

Bagian untuk menambahkan anggota dari institusi lain atau peneliti eksternal.

**Menambah Anggota**
1. Klik tombol **+ Tambah**
2. Akan tampil form untuk memasukkan data anggota:
   - NIK (Nomor Induk Kependudukan) atau identifikasi lainnya
   - Nama
   - Email
   - Tugas/peran dalam penelitian
3. Setelah mengisi, anggota akan muncul dalam tabel

**Tabel Anggota**
- Menampilkan kolom: No, NIK, Nama, Email, Tugas, Aksi
- Kolom Aksi untuk mengubah atau menghapus anggota

### 1.6 Identitas Mahasiswa

Bagian untuk menambahkan mahasiswa yang terlibat dalam penelitian.

**Menambah Mahasiswa**
1. Klik tombol **+ Tambah**
2. Akan tampil form untuk memasukkan data mahasiswa:
   - Jenis Anggota (S1, S2, S3, dll)
   - NIM (Nomor Induk Mahasiswa)
   - Nama
   - Tugas/peran dalam penelitian
   - Jumlah SKS (jika terkait dengan matakuliah)
   - Matakuliah (jika relevan)
3. Setelah mengisi, mahasiswa akan muncul dalam tabel

**Tabel Mahasiswa**
- Menampilkan kolom: No, Jenis Anggota, NIM, Nama, Tugas, Jumlah SKS, Matakuliah, Aksi
- Kolom Aksi untuk mengubah atau menghapus mahasiswa

### Melanjutkan ke Langkah 2

Setelah mengisi semua field wajib, klik tombol **Lanjutkan** untuk melanjutkan ke langkah berikutnya.

:::warning Penting
Tombol Lanjutkan hanya akan aktif jika semua field yang ditandai dengan (*) sudah diisi dengan benar. Periksa kembali data Anda sebelum melanjutkan.
:::

## Langkah 2-5: Langkah Selanjutnya

Wizard terdiri dari 5 langkah utama yang harus diselesaikan secara berurutan:

1. **IDENTITAS USULAN** - Informasi dasar dan tim penelitian (Langkah 1)
2. **SUBSTANSI USULAN** - Deskripsi, tujuan, dan metodologi penelitian
3. **RAB** - Rencana anggaran biaya
4. **DOKUMEN PENDUKUNG** - Upload dokumen pendukung
5. **KONFIRMASI USULAN** - Periksa dan konfirmasi semua data sebelum submit

Setiap langkah akan menjadi aktif setelah langkah sebelumnya berhasil diselesaikan.

:::info[Informasi Penting]

- Setiap langkah dapat disimpan dan dikerjakan kemudian (status DRAFT)
- Gunakan tombol "Kembali" untuk kembali ke langkah sebelumnya
- Data Anda otomatis tersimpan saat mengisi form
- Jangan lupa untuk menyelesaikan semua langkah sebelum mengajukan proposal

:::

## Menyimpan dan Melanjutkan Nanti

Jika Anda ingin mengerjakan proposal kemudian:

1. Anda dapat keluar dari wizard kapan saja
2. Proposal akan tersimpan dalam status **DRAFT**
3. Buka kembali proposal dari menu **Proposal Saya**
4. Klik proposal berstatus DRAFT untuk melanjutkan pengerjaan

## Mengajukan Proposal

Setelah menyelesaikan semua 5 langkah:

1. Lakukan review menyeluruh di langkah "KONFIRMASI USULAN"
2. Jika semua data sudah benar, klik tombol **Submit** atau **Kirim Proposal**
3. Proposal akan berubah status menjadi **SUBMITTED**
4. Anda akan menerima notifikasi bahwa proposal telah berhasil diajukan

:::warning Deadline
Pastikan Anda mengajukan proposal sebelum deadline batch yang Anda pilih. Proposal yang dikirim setelah deadline tidak akan diterima.
:::

## Troubleshooting

**Q: Tombol Lanjutkan tidak aktif padahal saya sudah mengisi semua field?**
A: Pastikan semua field yang ditandai dengan (*) sudah diisi dan sesuai dengan format yang diminta. Periksa juga tidak ada pesan error di bawah field input.

**Q: Bagaimana jika saya lupa password untuk mengakses sistem?**
A: Gunakan fitur "Lupa Password" di halaman login, atau hubungi administrator sistem.

**Q: Bisakah saya mengubah program dan batch setelah memulai wizard?**
A: Tidak. Program dan batch tidak dapat diubah setelah Anda mulai mengisi wizard. Jika perlu mengubah, Anda harus membuat proposal baru.
