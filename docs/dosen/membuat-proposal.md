# Membuat Proposal (7-step wizard)

Panduan lengkap untuk membuat proposal penelitian atau pengabdian kepada masyarakat melalui wizard 7 langkah di ORBIT UNSURYA.

## Memulai Pembuatan Proposal

1. Klik menu **Proposal** di sidebar
2. Pada halaman "Proposal Saya", klik tombol **Buat Proposal**
3. Anda akan diarahkan ke halaman pemilihan program dan batch

## Langkah 0: Pilih Program dan Batch

Sebelum memasuki wizard, Anda harus memilih program penelitian dan batch pengajuan.

![Pilih Program dan Batch](/img/dosen/proposal-create-step0-select-program.png)

### Memilih Program

1. Lihat daftar program penelitian yang tersedia di bagian "1. Select Program"
2. Setiap program menampilkan informasi:
   - Nama program
   - Jenis penelitian (Riset Dasar, Riset Terapan, dll)
   - Kategori (Penelitian, Pengabdian)
   - Dana maksimal
   - Jumlah batch aktif
3. Klik pada program yang sesuai dengan penelitian Anda
4. Program yang dipilih akan menampilkan tanda "Program selected"

### Memilih Batch

Setelah memilih program, batch yang tersedia akan ditampilkan di bagian "2. Select Batch".

1. Lihat daftar batch yang tersedia
2. Setiap batch menampilkan:
   - Nama batch
   - Deadline pengajuan
   - Sisa hari untuk pengajuan
   - Jumlah proposal yang sudah masuk
3. Klik batch yang ingin Anda gunakan
4. Batch yang dipilih akan menampilkan tanda "Batch selected"

### Lanjut ke Wizard

Setelah memilih program dan batch, tombol **Continue to Proposal Details** akan aktif.

![Batch Terpilih](/img/dosen/proposal-create-step0-batch-selected.png)

Klik tombol tersebut untuk masuk ke wizard 7 langkah.

:::warning[Perhatian]

Setelah Anda memilih program dan batch, tidak dapat mengubahnya selama proses pembuatan proposal. Pastikan pilihan sudah benar sebelum melanjutkan.

:::

## Langkah 1: Identitas Usulan

Langkah pertama adalah mengisi identitas dan informasi dasar proposal penelitian atau pengabdian.

![Identitas Usulan](/img/dosen/proposal-create-step1-identitas-usulan.png)

### 1.1 Identitas Penelitian

**1. Judul Proposal (Wajib)**
- Masukkan judul proposal yang jelas dan deskriptif
- Judul harus mencerminkan inti dari penelitian atau pengabdian yang akan dilakukan

### 1.2 Pemilihan Program Penelitian

**1. Bidang Fokus Penelitian (Wajib)**
- Pilih bidang fokus penelitian dari dropdown
- Opsi meliputi: Kedirgantaraan, Teknologi dan Rekayasa, Ekonomi dan Bisnis, Kesehatan dan Masyarakat, Pendidikan dan Sosial

**2. Tahun Pertama Usulan (Wajib)**
- Masukkan tahun dimulainya penelitian (minimum tahun 1970)
- Format: YYYY (contoh: 2025)

**3. Ruang Lingkup (Wajib)**
- Pilih dari dropdown: Nasional, Internasional, atau Lokal
- Ini menunjukkan jangkauan/skala penelitian Anda

**4. Lama Kegiatan (Wajib)**
- Durasi penelitian sudah terisi otomatis (default: 1 tahun)
- Dapat diubah sesuai kebutuhan

**5-7. Rumpun Ilmu (Wajib)**
- **Level 1 (Rumpun Ilmu)**: Pilih rumpun ilmu umum terlebih dahulu
- **Level 2 (Sub Rumpun)**: Akan aktif setelah Level 1 dipilih
- **Level 3 (Bidang Ilmu)**: Akan aktif setelah Level 2 dipilih
- Hierarki ini memastikan klasifikasi penelitian yang tepat sesuai taksonomi universitas

**Taksonomi Bidang Fokus (Opsional)**
- Bagian tambahan untuk memperdalam klasifikasi penelitian
- Mencakup: Bidang Fokus Penelitian, Tema Riset, Topik Riset
- Hubungan antar field sama dengan Rumpun Ilmu - bersifat bertingkat

### 1.3 Identitas Ketua

**Nama Ketua**
- Nama otomatis terisi dari profil Anda
- Dapat diubah jika diperlukan

**Uraian Tugas dalam Penelitian (Wajib)**
- Jelaskan peran dan tanggung jawab Anda sebagai ketua penelitian
- Contoh: "Mengkoordinasikan dan mengelola keseluruhan pelaksanaan penelitian"

### 1.4 Identitas Anggota Penelitian (Dosen)

Bagian untuk menambahkan dosen yang terlibat dalam penelitian.

**Menambah Anggota**
1. Klik tombol **+ Tambah**
2. Akan tampil form untuk memasukkan data anggota dosen:
   - NIDN (Nomor Induk Dosen Nasional)
   - Nama
   - Tugas/peran dalam penelitian
3. Setelah mengisi, anggota akan muncul dalam tabel

**Tabel Anggota**
- Menampilkan kolom: No, NIDN, Nama, Tugas, Status, Aksi
- Status menunjukkan verifikasi anggota
- Kolom Aksi untuk mengubah atau menghapus anggota

:::tip[Catatan]

Anggota dosen harus merupakan dosen Universitas Dirgantara Marsekal Suryadarma yang terdaftar dalam sistem.

:::

### 1.5 Identitas Anggota Non Dosen Unsurya

Bagian untuk menambahkan anggota dari institusi lain atau peneliti eksternal.

**Menambah Anggota**
1. Klik tombol **+ Tambah**
2. Akan tampil form untuk memasukkan data anggota:
   - NIK (Nomor Induk Kependudukan) atau identifikasi lainnya
   - Nama
   - Email
   - Tugas/peran dalam penelitian
3. Setelah mengisi, anggota akan muncul dalam tabel

**Tabel Anggota**
- Menampilkan kolom: No, NIK, Nama, Email, Tugas, Aksi
- Kolom Aksi untuk mengubah atau menghapus anggota

### 1.6 Identitas Mahasiswa

Bagian untuk menambahkan mahasiswa yang terlibat dalam penelitian.

**Menambah Mahasiswa**
1. Klik tombol **+ Tambah**
2. Akan tampil form untuk memasukkan data mahasiswa:
   - Jenis Anggota (S1, S2, S3, dll)
   - NIM (Nomor Induk Mahasiswa)
   - Nama
   - Tugas/peran dalam penelitian
   - Jumlah SKS (jika terkait dengan matakuliah)
   - Matakuliah (jika relevan)
3. Setelah mengisi, mahasiswa akan muncul dalam tabel

**Tabel Mahasiswa**
- Menampilkan kolom: No, Jenis Anggota, NIM, Nama, Tugas, Jumlah SKS, Matakuliah, Aksi
- Kolom Aksi untuk mengubah atau menghapus mahasiswa

### Melanjutkan ke Langkah 2

Setelah mengisi semua field wajib, klik tombol **Lanjutkan** untuk melanjutkan ke langkah berikutnya.

:::warning[Penting]

Tombol Lanjutkan hanya akan aktif jika semua field yang ditandai dengan (*) sudah diisi dengan benar. Periksa kembali data Anda sebelum melanjutkan.

:::

## Langkah 2: Substansi Usulan

Langkah ini untuk menambahkan mitra dan mengunggah dokumen substansi proposal Anda.

![Substansi Usulan](/img/dosen/proposal-create-step2-substansi.png)

### 2.1 Identitas Mitra

Bagian ini menampilkan daftar mitra yang terlibat dalam proposal (opsional).

**Menambah Mitra**
1. Klik tombol **+ Tambah** untuk menambahkan mitra baru
2. Akan tampil form untuk memasukkan data mitra:
   - Nama Mitra
   - Instansi/Organisasi
   - Bidang Usaha
3. Setelah mengisi, mitra akan muncul dalam tabel

**Tabel Mitra**
- Menampilkan kolom: No, Mitra, Instansi, Bidang Usaha, Aksi
- Kolom Aksi untuk mengubah atau menghapus mitra

:::note[Catatan]

Penambahan mitra bersifat opsional. Jika proposal Anda tidak melibatkan mitra eksternal, bagian ini dapat dikosongkan.

:::

### 2.2 Substansi Usulan

**Dokumen Substansi (Wajib)**
1. Klik tombol **Download Template** untuk mengunduh template dokumen substansi
2. Isi template dengan penjelasan lengkap tentang:
   - Tujuan penelitian/pengabdian
   - Metodologi yang akan digunakan
   - Jadwal pelaksanaan
   - Luaran yang diharapkan
3. Unggah file PDF yang sudah diisi ke area upload
   - Format: PDF
   - Ukuran maksimal: 10 MB

:::warning[Penting]

Dokumen substansi harus jelas dan terstruktur. Reviewer akan mengevaluasi kualitas ide dan metodologi penelitian Anda berdasarkan bagian ini.

:::

### Melanjutkan ke Langkah 3

Setelah mengisi substansi usulan, klik tombol **Lanjutkan** untuk melanjutkan ke langkah Rancangan Anggaran Biaya (RAB).

## Langkah 3: Rancangan Anggaran Biaya (RAB)

Langkah ini untuk merencanakan dan mengalokasikan anggaran untuk penelitian atau pengabdian Anda.

![Rancangan Anggaran Biaya](/img/dosen/proposal-create-step3-rab.png)

### 3.1 Rencana Anggaran Belanja

**Menambah Item RAB (Wajib)**
1. Klik tombol **+ Tambah** untuk menambahkan item anggaran
2. Akan tampil form untuk memasukkan:
   - Kelompok RAB (misalnya: Tenaga Kerja, Bahan, Peralatan, Perjalanan, dll)
   - Komponen (nama komponen, misalnya: Gaji peneliti, Bahan kimia, dll)
   - Item (detail item, misalnya: Peneliti utama, Bahan X jenis Y)
   - Satuan (unit, misalnya: orang, kg, buah, dll)
   - Volume (jumlah)
   - Harga Satuan (harga per unit)
   - Total (otomatis dihitung: volume × harga satuan)
3. Setelah mengisi, item akan muncul dalam tabel

**Tabel RAB**
- Menampilkan kolom: Kelompok RAB, Komponen, Item, Satuan, Volume, Harga Satuan, Total, Aksi
- Total anggaran otomatis dihitung dari semua item
- Kolom Aksi untuk mengubah atau menghapus item

:::warning[Penting]

Pastikan total anggaran yang Anda usulkan tidak melebihi batas maksimal program yang ditampilkan pada halaman pilih program. Anggaran harus realistis dan dapat dipertanggungjawabkan.

:::

### 3.2 Catatan (Opsional)

**Tambahan Catatan untuk Anggaran**
- Anda dapat menambahkan catatan atau penjelasan tambahan tentang rencana anggaran
- Gunakan textarea untuk menjelaskan dasar perhitungan atau hal penting lainnya
- Bagian ini bersifat opsional

### Melanjutkan ke Langkah 4

Setelah mengisi semua item RAB, klik tombol **Lanjutkan** untuk melanjutkan ke langkah Dokumen Pendukung.

## Langkah 4: Dokumen Pendukung

Langkah ini untuk mengunggah dokumen-dokumen pendukung yang diperlukan untuk proposal Anda.

![Dokumen Pendukung](/img/dosen/proposal-create-step4-dokumen.png)

### 4.1 Data Pendukung

Halaman ini menampilkan dokumen-dokumen wajib dan opsional yang harus atau dapat diupload.

**1. Surat Pernyataan Orisinalitas (Wajib)**
- **Deskripsi**: Surat Pernyataan Orisinalitas Usulan yang ditandatangani oleh ketua pelaksana dan bermeterai Rp10.000,00
- **Status**: Wajib diupload
- **Template**: Klik tombol **Download Template** untuk mengunduh template surat
- **Cara Upload**:
  1. Isi template surat sesuai data proposal Anda
  2. Tanda tangani dan beri meterai Rp10.000,00
  3. Scan atau ambil foto dokumen yang sudah ditanda tangani
  4. Unggah file PDF ke area upload
- **Format & Ukuran**: PDF, maksimal 10 MB

**2. Dokumen Dukung Lainnya (Opsional)**
- Anda dapat mengunggah dokumen pendukung tambahan seperti:
  - CV peneliti utama
  - Surat dukungan dari institusi mitra
  - Izin etik (jika diperlukan)
  - Dokumen lain yang relevan
- **Format & Ukuran**: PDF, maksimal 10 MB per dokumen

:::warning[Penting]

- Surat Pernyataan Orisinalitas **wajib** diupload sebelum melanjutkan ke langkah berikutnya
- Pastikan scan/foto dokumen jelas dan dapat dibaca
- Gunakan format PDF untuk konsistensi

:::

### Melanjutkan ke Langkah 5

Setelah mengunggah semua dokumen yang diperlukan, klik tombol **Lanjutkan** untuk melanjutkan ke langkah Konfirmasi Usulan.

## Langkah 5: Konfirmasi Usulan

Langkah terakhir adalah mereviu semua data proposal Anda sebelum pengajuan.

![Konfirmasi Usulan](/img/dosen/proposal-create-step5-konfirmasi.png)

### 5.1 Review Proposal

**Ringkasan Lengkap**
Halaman ini menampilkan ringkasan lengkap dari semua data yang telah Anda isi di langkah 1-4, disusun dalam beberapa bagian:

**1. Identitas Usulan**
- Judul Proposal
- Kategori Program
- Bidang Fokus Penelitian
- Kelompok Skema
- Ruang Lingkup
- Tahun Usulan
- Lama Kegiatan
- Ketua Pelaksana

**2. Tim Pelaksana**
- Ketua Pelaksana
- Daftar Anggota Dosen (jika ada)
- Daftar Anggota Non-Dosen (jika ada)
- Daftar Mahasiswa (jika ada)

**3. Identitas Mitra (jika ada)**
- Daftar mitra yang ditambahkan di langkah 2

**4. Substansi Usulan**
- Dokumen substansi yang sudah diupload

**5. Rancangan Anggaran Biaya**
- Daftar item RAB
- Total anggaran yang diminta

**6. Dokumen Pendukung**
- Surat Pernyataan Orisinalitas
- Dokumen pendukung lainnya (jika ada)

**Periksa Kembali Data**
1. Baca setiap bagian dengan teliti
2. Pastikan semua informasi sudah benar
3. Verifikasi bahwa semua dokumen sudah terupload dengan baik
4. Jika ada kesalahan, gunakan tombol **Ubah** di bagian yang perlu diperbaiki

### 5.2 Tombol Aksi

**Tombol Ubah**
- Tekan tombol ini untuk kembali memperbaiki data di langkah tertentu
- Sistem akan membuka langkah yang ingin Anda ubah
- Setelah selesai mengubah, Anda akan kembali ke halaman konfirmasi

**Tombol Submit Proposal**
- Tombol ini digunakan untuk mengajukan proposal secara resmi
- Pastikan semua data sudah benar sebelum mengklik tombol ini
- Setelah disubmit, proposal tidak dapat diubah lagi

:::warning[Perhatian - Periksa Sebelum Submit]

Sebelum mengklik **Submit Proposal**, pastikan:
- Semua data di setiap bagian sudah lengkap dan benar
- Semua dokumen wajib (terutama Surat Pernyataan Orisinalitas) sudah diupload
- Proposal akan diajukan sebelum deadline batch
- Anda telah membuat salinan data proposal Anda (jika diperlukan)

**Catatan Penting**: Setelah proposal disubmit, Anda TIDAK dapat mengubah data lagi. Jika ada kesalahan serius setelah submit, hubungi administrator untuk bantuan revisi.

:::

### Setelah Pengajuan

Setelah proposal berhasil disubmit:
1. Status proposal berubah menjadi **SUBMITTED**
2. Anda akan menerima notifikasi email konfirmasi
3. Proposal akan masuk ke tahap evaluasi selanjutnya
4. Pantau status proposal di halaman **Proposal Saya**

## Menyimpan dan Melanjutkan Nanti

Jika Anda ingin mengerjakan proposal kemudian:

1. Anda dapat keluar dari wizard kapan saja
2. Proposal akan tersimpan dalam status **DRAFT**
3. Buka kembali proposal dari menu **Proposal Saya**
4. Klik proposal berstatus DRAFT untuk melanjutkan pengerjaan

## Troubleshooting

**Q: Tombol Lanjutkan tidak aktif padahal saya sudah mengisi semua field?**
A: Pastikan semua field yang ditandai dengan (*) sudah diisi dan sesuai dengan format yang diminta. Periksa juga tidak ada pesan error di bawah field input.

**Q: Bagaimana jika saya lupa password untuk mengakses sistem?**
A: Gunakan fitur "Lupa Password" di halaman login, atau hubungi administrator sistem.

**Q: Bisakah saya mengubah program dan batch setelah memulai wizard?**
A: Tidak. Program dan batch tidak dapat diubah setelah Anda mulai mengisi wizard. Jika perlu mengubah, Anda harus membuat proposal baru.
