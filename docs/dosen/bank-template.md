# Bank Template

Halaman ini menjelaskan cara mengakses dan mengunduh template dokumen yang tersedia di ORBIT UNSURYA. Template mencakup dokumen proposal, laporan kemajuan, laporan akhir, dan dokumen pendukung lainnya.

## Cara Membuka Halaman Bank Template

Klik menu **Bank Template** di sidebar kiri, di bawah bagian **Pengaturan Program**.

![Halaman Bank Template](/img/dosen/bank-template-list.png)

Halaman menampilkan daftar template dengan kolom berikut:

| Kolom | Keterangan |
|-------|------------|
| No | Nomor urut template |
| Judul | Nama template dengan ikon jenis file |
| Deskripsi | Keterangan singkat isi template |
| Tanggal Upload | Tanggal template diunggah ke sistem |
| Aksi | Tombol **Download** untuk mengunduh file |

Di bagian bawah tabel terdapat informasi pagination, contoh: _"Menampilkan 1–4 dari 4 template"_, beserta tombol navigasi halaman.

## Mencari Template

Gunakan kolom pencarian **"Cari template..."** di atas tabel untuk menemukan template tertentu.

![Hasil Pencarian Template](/img/dosen/bank-template-search.png)

1. Ketik kata kunci pada kolom pencarian (contoh: "laporan")
2. Hasil pencarian akan muncul secara otomatis
3. Daftar template dan informasi jumlah hasil akan diperbarui sesuai kata kunci

:::info[Pencarian]

Pencarian berdasarkan judul template dan tidak membedakan huruf besar/kecil.

:::

## Mengunduh Template

1. Temukan template yang ingin diunduh dari daftar
2. Klik tombol **Download** (ikon unduh) pada kolom Aksi di baris template
3. File akan otomatis terunduh melalui browser Anda

:::tip[Format File]

Template tersedia dalam format PDF (.pdf) dan Microsoft Word (.doc/.docx). Pastikan perangkat Anda memiliki aplikasi yang sesuai untuk membuka file yang diunduh.

:::
