# Rekap Laporan Akhir

Halaman ini menjelaskan cara memantau rekapitulasi laporan akhir dari seluruh proposal yang telah didanai, terkelompok berdasarkan skema atau program studi.

## Membuka Halaman Rekap Laporan Akhir

1. Login ke ORBIT UNSURYA sebagai **Admin**
2. Di sidebar kiri, pada bagian **Proses Proposal**, klik **Rekap Laporan Akhir**
3. Sistem akan menampilkan tabel rekapitulasi laporan akhir yang dikelompokkan per skema

## Daftar Rekapitulasi Laporan Akhir

Halaman utama menampilkan ringkasan laporan akhir per skema atau per program studi.

![Daftar Rekap Laporan Akhir](/img/admin/final-report-recap-list.png)

| Kolom | Keterangan |
|---|---|
| **No.** | Nomor urut |
| **Skema** | Nama skema atau program studi (tergantung pilihan pengelompokan) |
| **Total Didanai** | Jumlah proposal yang didanai dalam skema tersebut |
| **Total Lap. Akhir** | Jumlah laporan akhir yang sudah diunggah |
| **Status** | Persentase proposal yang sudah mengumpulkan laporan akhir |
| **Aksi** | Tombol **Detail** untuk melihat daftar proposal per skema |

## Memfilter Data

Bagian atas halaman menyediakan beberapa filter untuk menyaring tampilan.

![Filter Rekap Laporan Akhir](/img/admin/final-report-recap-filter.png)

| Filter | Keterangan |
|---|---|
| **Jenis Kegiatan** | Menyaring berdasarkan jenis kegiatan (Penelitian atau Pengabdian) |
| **Skema** | Menyaring berdasarkan nama skema tertentu |
| **Tahun Pelaksanaan** | Menyaring berdasarkan tahun pelaksanaan proposal |

### Kelompokkan Berdasarkan

Di bagian bawah filter terdapat opsi **Kelompokkan berdasarkan** dengan dua pilihan:

- **Skema** — data dikelompokkan per skema program (dipilih secara default)
- **Prodi** — data dikelompokkan per program studi

## Mengekspor Data

Untuk mengunduh data rekapitulasi laporan akhir:

1. Klik tombol **Export** di pojok kanan atas halaman
2. File akan diunduh secara otomatis ke perangkat Anda

## Melihat Detail per Skema

Untuk melihat daftar proposal laporan akhir dalam skema tertentu:

1. Pada baris skema yang ingin diperiksa, klik tombol **Detail** di kolom Aksi
2. Sistem akan membuka halaman **Detail Rekap Laporan Akhir**

### Halaman Detail

![Detail Rekap Laporan Akhir](/img/admin/final-report-recap-detail.png)

Halaman detail menampilkan daftar semua proposal dalam skema terpilih beserta status pengumpulan laporan akhirnya.

| Kolom | Keterangan |
|---|---|
| **No.** | Nomor urut |
| **Nama Pengusul** | Nama dosen pengusul proposal |
| **Tanggal** | Tanggal pengumpulan laporan akhir (ditampilkan jika sudah diunggah) |
| **Judul** | Judul proposal |
| **Status** | Status laporan akhir: *Belum Unggah* atau status lainnya sesuai progres |
| **Aksi** | Aksi yang tersedia sesuai status laporan |

Gunakan selector **Rows per page** di bagian kanan atas tabel untuk mengatur jumlah data yang ditampilkan per halaman.

Klik tombol **Kembali ke Rekap** untuk kembali ke halaman utama Rekap Laporan Akhir.
