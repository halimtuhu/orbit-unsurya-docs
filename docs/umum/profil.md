# Profil

Halaman **Profil** dapat diakses oleh semua role (Admin, Dosen, dan Reviewer) untuk melihat dan memperbarui informasi akun pribadi, serta mengganti kata sandi.

## Cara Membuka Halaman Profil

1. Klik tombol **Menu pengguna** di pojok kanan atas halaman (menampilkan inisial nama Anda, misalnya **AA**, **DD**)

2. Akan muncul dropdown menu — klik **Profil**

3. Sistem akan membuka halaman **Profil Saya**

![Halaman Profil Saya](/img/umum/profil-main.png)

## Informasi yang Ditampilkan

Halaman profil menampilkan kartu profil berisi:

| Informasi | Keterangan |
|-----------|------------|
| Nama | Nama lengkap akun Anda |
| Email | Alamat email yang terdaftar |
| Roles | Role yang dimiliki di sistem (misalnya: admin, dosen, reviewer) |

## Mengedit Data Diri

1. Klik tombol **ikon pensil** di pojok kanan atas kartu profil

2. Dialog **Edit Profil** akan muncul

3. Perbarui informasi yang diperlukan:

![Form Edit Profil](/img/umum/profil-edit-form.png)

| Field | Keterangan |
|-------|------------|
| Nama Lengkap | Wajib diisi. Nama lengkap Anda |
| Email | Opsional. Alamat email aktif |
| NIDN | Opsional. Nomor Induk Dosen Nasional, minimal 8 digit angka |
| NUPTK | Opsional. Nomor Unik Pendidik dan Tenaga Kependidikan, harus 16 digit angka |
| NIP | Opsional. Nomor Induk Pegawai, harus 18 digit angka |

4. Klik **Simpan Perubahan** untuk menyimpan, atau **Batal** untuk membatalkan

:::info[Informasi]

Field NIDN, NUPTK, dan NIP bersifat opsional dan disesuaikan dengan jabatan Anda di institusi.

:::

## Mengganti Password

1. Klik tombol **Ganti Password** di bagian bawah kartu profil

2. Dialog **Ganti Password** akan muncul

3. Isi ketiga field yang tersedia:

![Form Ganti Password](/img/umum/profil-ganti-password.png)

| Field | Keterangan |
|-------|------------|
| Password Lama | Password Anda yang saat ini aktif |
| Password Baru | Password baru yang ingin digunakan, minimal 6 karakter |
| Konfirmasi Password Baru | Ketik ulang password baru untuk konfirmasi |

:::tip[Tips]

Gunakan ikon mata di setiap field untuk menampilkan atau menyembunyikan karakter password.

:::

4. Klik **Ganti Password** untuk menyimpan perubahan, atau **Batal** untuk membatalkan

:::warning[Perhatian]

Pastikan Anda mengingat password baru sebelum menyimpan. Setelah password diganti, gunakan password baru untuk login berikutnya.

:::
